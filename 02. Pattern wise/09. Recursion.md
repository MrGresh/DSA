# Recursion
Solving problems by breaking them down into smaller, self-similar subproblems.

---

# Linear Recursion
Solve problem by reducing to size n-1 → combine results linearly.

| # | Problem | Difficulty |
|---|---|---|
| 1 | Factorial | 🟢 Easy |
| 2 | Fibonacci | 🟢 Easy |
| 3 | Power of x | 🟡 Medium |

---

# Divide & Conquer
Break problem into independent subproblems → solve recursively → combine results.

| # | Problem | Difficulty |
|---|---|---|
| 1 | Binary Search | 🟢 Easy |
| 2 | Merge Sort | 🟡 Medium |
| 3 | Maximum Subarray | 🟡 Medium |
| 4 | Search in Rotated Sorted Array | 🟡 Medium |
