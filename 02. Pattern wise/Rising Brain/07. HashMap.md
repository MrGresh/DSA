# HashMap
Key-value pair data structure for O(1) average time complexity lookups.

---

# Frequency Map / Counting
Count elements to find majority, top-k frequent, or sort by frequency.

| # | Problem | Difficulty |
|---|---|---|
| 1 | Majority Element | 🟢 Easy |
| 2 | Top K Frequent Elements | 🟡 Medium |
| 3 | Sort Characters By Frequency | 🟡 Medium |
| 4 | Task Scheduler (frequency-based greedy) | 🟡 Medium |

# Prefix-Sum with Map
Track cumulative sums; map stores first occurrence → solve subarray sum problems.

| # | Problem | Difficulty |
|---|---|---|
| 1 | Subarray Sum Equals K | 🟡 Medium |
| 2 | Continuous Subarray Sum | 🟡 Medium |
| 3 | Subarray Sums Divisible by K | 🟡 Medium |
| 4 | Count Subarrays with Sum K | 🟡 Medium |
