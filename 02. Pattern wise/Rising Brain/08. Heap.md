# Heap
Priority Queue data structure for efficient retrieval of highest/lowest priority elements.

---

# Top-K Elements
Use min-heap for top-k largest, max-heap for top-k smallest → maintain heap of size k.

| # | Problem | Difficulty |
|---|---|---|
| 1 | K Frequent Words | 🟡 Medium |
| 2 | Sort Characters By Frequency | 🟡 Medium |
| 3 | Kth Largest Element in an Array | 🟡 Medium |
| 4 | Top K Frequent Elements | 🟡 Medium |

| 6 | Minimum Cost to Connect Ropes | 🟡 Medium |
