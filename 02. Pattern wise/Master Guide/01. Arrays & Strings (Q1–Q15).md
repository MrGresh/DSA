# 🧊 Section 1: Arrays & Strings (Q1–Q15)

**Core Concept:** Arrays and Strings are the backbone of 40%+ of interview questions. 

**Key Patterns:** Prefix sums, in-place manipulation, two-pointers, and sliding windows. 

**For Strings:** Character frequency maps (HashMaps/int arrays), Stacks, or Sliding Windows.

---

### 🎯 Pattern Triggers
| If you see... | Use this Pattern... |
| :--- | :--- |
| "Find subarray/substring with X property" | 🪟 **Sliding Window** |
| "Find two elements satisfying condition in sorted array" | 👈 **Two Pointers** 👉 |
| "Count subarrays with sum = K" | 🔢 **Prefix Sum + HashMap** |
| "Rearrange in-place" | 🚩 **Dutch National Flag / Partition** |

---

### 📋 Essential Problem Set

| # | Question | Key Concept | Approach | Complexity (T/S) |
| :-- | :--- | :--- | :--- | :--- |
| 1 | **Two Sum** | HashMap | Store complement; one-pass $O(n)$ | $O(n) / O(n)$ |
| 2 | **Best Time to Buy/Sell Stock** | Greedy | Track min so far, update max profit | $O(n) / O(1)$ |
| 3 | **Contains Duplicate** | HashSet | Add to set; return true if already present | $O(n) / O(n)$ |
| 4 | **Product of Array Except Self** | Prefix & Suffix | Left pass then right pass, no division | $O(n) / O(1)$ |
| 5 | **Maximum Subarray (Kadane's)** | DP | $dp[i] = \max(nums[i], dp[i-1] + nums[i])$ | $O(n) / O(1)$ |
| 6 | **Maximum Product Subarray** | DP | Track both max and min at each step | $O(n) / O(1)$ |
| 7 | **Find Min in Rotated Sorted Array** | Binary Search | Compare mid with right to decide direction | $O(\log n) / O(1)$ |
| 8 | **Search in Rotated Sorted Array** | Binary Search | Determine sorted half, narrow range | $O(\log n) / O(1)$ |
| 9 | **3Sum** | Sort + 2 Pointers | Sort, fix one element, two-pointer for rest | $O(n^2) / O(1)$ |
| 10 | **Container With Most Water** | Two Pointers | Move pointer with smaller height inward | $O(n) / O(1)$ |
| 11 | **Longest Substring w/o Repeating** | Sliding Window | Expand right, shrink left on duplicate | $O(n) / O(n)$ |
| 12 | **Longest Repeating Char Replace** | Sliding Window | Window size - $max\_freq \le k$ | $O(n) / O(1)$ |
| 13 | **Minimum Window Substring** | Sliding Window | Two maps: need vs have; shrink when valid | $O(n) / O(n)$ |
| 14 | **Valid Anagram** | Freq Count | Count chars in $s$, decrement with $t$ | $O(n) / O(1)$ |
| 15 | **Group Anagrams** | HashMap | Sort each string as key, group by key | $O(nk \log k) / O(nk)$ |

---
*Note: $k$ in Group Anagrams refers to the maximum length of a string.*
