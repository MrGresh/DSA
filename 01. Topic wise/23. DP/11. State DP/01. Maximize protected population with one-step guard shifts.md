# Amazon OA: City Management System (Security Unit Relocation)

## Problem Description
A city management system oversees $n$ cities, each with a specific population. To ensure safety, some cities are equipped with security units. You are tasked with relocating these units to maximize the total protected population.

### Input
1.  **population[]**: An integer array of size $n$, where `population[i]` is the number of inhabitants in city $i$.
2.  **unit**: A binary string of length $n$, where `unit[i] = '1'` means city $i$ has a security unit, and `'0'` means it does not.

### Relocation Rules
* **One-Step Left:** A security unit at city $i$ (where $i > 1$) can be moved one step to the left to city $i-1$.
* **Single Move:** Each unit can be moved at most once.
* **Transfer:** If a unit is moved, city $i$ loses its unit and city $i-1$ gains one.
* **Boundary:** The security unit at city 1 cannot be moved further left.
* **Protection:** A city is considered **protected** if it has at least one security unit after all relocations are complete.

**Goal:** Determine the maximum total population that can be protected by optimally relocating the security units.

---

### Constraints
* $1 \le n \le 10^5$
* $1 \le population[i] \le 10^4$
* `unit` is a binary string of '0's and '1's.
* **Indexing:** The problem uses 1-based indexing for cities.

---

### Example
**Input:**
`n = 5`
`pop = [10, 5, 8, 9, 6]`
`unit = "01101"`

**Output:** `27`

**Explanation:**
1.  Initial state: `01101` (Cities 2, 3, and 5 have units).
2.  Move unit from City 2 to City 1: `10101` (Protected pop: $10+8+6 = 24$).
3.  Move unit from City 5 to City 4: `10110` (Protected pop: $10+8+9 = 27$).
4.  Maximum population protected is **27**.
### Memoization
```java
class Solution {
    private Long[][] memo;

    public long moveUnits(List<Integer> pop, String unit) {
        int n = pop.size();
        memo = new Long[n][2];
        return solve(n - 1, 0, pop, unit);
    }

    private long solve(int i, int isFilledByRight, List<Integer> pop, String unit) {
        if (i < 0) return 0;
        if (memo[i][isFilledByRight] != null) return memo[i][isFilledByRight];

        long res = 0;
        boolean hasUnit = (unit.charAt(i) == '1');

        if (isFilledByRight == 1) {
            res = pop.get(i) + solve(i - 1, (hasUnit && i > 0) ? 1 : 0, pop, unit);
        } else {
            if (hasUnit) {
                long stay = pop.get(i) + solve(i - 1, 0, pop, unit);
                long moveLeft = (i > 0) ? solve(i - 1, 1, pop, unit) : 0;
                res = Math.max(stay, moveLeft);
            } else res = solve(i - 1, 0, pop, unit);
        }

        return memo[i][isFilledByRight] = res;
    }
}
```
### Tabulation
```java
long moveUnits(List<Integer> pop, String unit) {
    int n = pop.size();
    long[][] dp = new long[n][2];
    for(int i=0;i<n;i++) {
        boolean hasUnit = (unit.charAt(i) == '1');
        for(int isFilledByRight = 0; isFilledByRight <= 1; isFilledByRight++) {
            long res = 0;

            if (isFilledByRight == 1) {
                res = pop.get(i) + ((i>0) ? dp[i - 1][hasUnit ? 1 : 0] : 0);
            } else {
                if (hasUnit) {
                    long stay = pop.get(i) + ((i>0) ? dp[i - 1][0] : 0);
                    long moveLeft = ((i>0) ? dp[i - 1][1] : 0);
                    res = Math.max(stay, moveLeft);
                } else res = ((i>0) ? dp[i - 1][0] : 0);
            }

            dp[i][isFilledByRight] = res;
        }
    }
    return dp[n-1][0];
}
```
### Space Optimization
```java
long moveUnits(List<Integer> pop, String unit) {
    int n = pop.size();
    long[] prev = new long[2];
    for(int i=0;i<n;i++) {
        long[] curr = new long[2];
        boolean hasUnit = (unit.charAt(i) == '1');
        for(int isFilledByRight = 0; isFilledByRight <= 1; isFilledByRight++) {
            long res = 0;

            if (isFilledByRight == 1) {
                res = pop.get(i) + ((i>0) ? prev[hasUnit ? 1 : 0] : 0);
            } else {
                if (hasUnit) {
                    long stay = pop.get(i) + ((i>0) ? prev[0] : 0);
                    long moveLeft = ((i>0) ? prev[1] : 0);
                    res = Math.max(stay, moveLeft);
                } else res = ((i>0) ? prev[0] : 0);
            }

            curr[isFilledByRight] = res;
        }
        prev = curr;
    }
    return prev[0];
}
```
