### BFS (The Standard)
**Strategy**: BFS is the natural choice because it explores the graph in "shells" or "layers." The first time a "wave" originating from source $i$ hits a node it has already visited, that path is guaranteed to be the shortest cycle involving i.
```java
public int findShortestCycleBFS(int n, List<List<Integer>> adj) {
    int minCycle = Integer.MAX_VALUE;

    // We must try starting from every node to find the global shortest cycle
    for (int i = 0; i < n; i++) {
        int[] dist = new int[n];
        Arrays.fill(dist, -1);
        
        Queue<Integer> q = new LinkedList<>();
        q.add(i);
        dist[i] = 0;

        while (!q.isEmpty()) {
            int u = q.poll();

            for (int v : adj.get(u)) {
                if (dist[v] == -1) {
                    // First time visiting v from this source i
                    dist[v] = dist[u] + 1;
                    q.add(v);
                } else {
                    // v was already visited! We found a cycle.
                    // Since BFS is level-order, this is the shortest cycle for source i.
                    minCycle = Math.min(minCycle, dist[u] - dist[v] + 1);
                    
                    // Optimization: Once a cycle is found for source i, 
                    // no need to continue this specific BFS.
                    break; 
                }
            }
            // If we found a triangle (3), it's the absolute minimum; we can stop everything.
            if (minCycle == 3) return 3;
        }
    }
    return minCycle == Integer.MAX_VALUE ? -1 : minCycle;
}
```
### DFS (The Alternative)
**Note:** To find the shortest cycle with DFS, we have to use backtracking. Unlike the BFS version, DFS doesn't explore level-by-level, so just finding the first back-edge isn't enough—it might be a long one! We have to explore all paths but "prune" (stop) any path that gets longer than the shortest cycle we've already found.
```java
class Solution {
    int minCycle = Integer.MAX_VALUE;

    public int findShortestCycleDFS(int n, List<List<Integer>> adj) {
        int[] dist = new int[n];
        Arrays.fill(dist, -1);

        for (int i = 0; i < n; i++) {
            // We use a fresh distance array for each starting point
            // to ensure we find the shortest path from THAT specific start.
            Arrays.fill(dist, -1);
            dfs(i, 0, dist, adj);
            if (minCycle == 3) return 3; // Global minimum check
        }
        return minCycle == Integer.MAX_VALUE ? -1 : minCycle;
    }

    private void dfs(int u, int d, int[] dist, List<List<Integer>> adj) {
        // Optimization: If current depth is already >= minCycle, no point continuing
        if (d >= minCycle) return;

        dist[u] = d;

        for (int v : adj.get(u)) {
            if (dist[v] != -1) {
                // Cycle detected!
                int cycleLen = d - dist[v] + 1;
                minCycle = Math.min(minCycle, cycleLen);
            } else {
                dfs(v, d + 1, dist, adj);
            }
        }

        // BACKTRACK: Reset the distance so this node can be 
        // visited via a different (potentially shorter) path later.
        dist[u] = -1;
    }
}
```
