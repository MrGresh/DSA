# 🚀 Graph Cycle Theory: The Ultimate Cheat Sheet

A comprehensive guide to finding shortest, longest, and weighted cycles using DFS and BFS in both Directed and Undirected graphs.

---

## 📌 Core Logic Selection
| Goal | Best Tool | Why? |
| :--- | :--- | :--- |
| **Shortest Cycle (Girth)** | **BFS** | Explores level-by-level; finds the "closest" meeting point first. |
| **Longest Cycle** | **DFS** | Dives deep into paths; naturally suited for "depth." |
| **Cycle Detection** | **DFS / BFS** | Directed: `recStack` (DFS) or `Kahn's` (BFS). Undirected: `parent` check. |
| **Max Weight Cycle** | **DFS + Backtrack** | NP-Hard. Requires exploring all path combinations (Small $N$ only). |

---

## 🛠️ DFS Strategies Comparison

| Goal | Reset `vis` per Source? | Backtrack? | Complexity |
| :--- | :--- | :--- | :--- |
| **Any Cycle** | **No** | **No** | $O(V + E)$ |
| **Longest Cycle** | **No** | **No** | $O(V + E)$ |
| **Shortest Cycle** | **Yes** | **Yes** | $O(V \cdot \text{Paths})$ |
| **Max Weight Cycle** | **Yes** | **Yes** | $O(V \cdot \text{Paths})$ |

> **Note:** "Backtracking" means setting `dist[u] = -1` and `inPath[u] = false` after the recursive calls return.

## 📐 Cycle Length Formulas

| Graph Type | Algorithm | Formula | Key Condition |
| :--- | :--- | :--- | :--- |
| **Directed** | **BFS** | `dist[u] - dist[v] + 1` | First visited node `v` |
| **Directed** | **DFS** | `dist[u] - dist[v] + 1` | `v` is in `recStack` |
| **Undirected** | **BFS** | `dist[u] + dist[v] + 1` | `v != parent[u]` |
| **Undirected** | **DFS** | `currDepth - dist[v] + 1` | `v != parent` |
