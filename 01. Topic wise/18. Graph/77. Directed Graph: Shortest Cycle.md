### BFS
**Strategy**: BFS from every node. Since BFS explores level-by-level, the first time we see a node we've already visited in the same BFS run, it forms the shortest cycle for that starting node.
```java
class DirectedShortest {
    public int find(int V, List<List<Integer>> adj) {
        int minCycle = Integer.MAX_VALUE;
        for (int i = 0; i < V; i++) {
            int[] dist = new int[V];
            Arrays.fill(dist, -1);
            Queue<Integer> q = new LinkedList<>();
            q.add(i); dist[i] = 0;
            while (!q.isEmpty()) {
                int u = q.poll();
                for (int v : adj.get(u)) {
                    if (dist[v] == -1) {
                        dist[v] = dist[u] + 1;
                        q.add(v);
                    } else { // In directed, any visited node is a cycle
                        minCycle = Math.min(minCycle, dist[u] + dist[v] + 1);
                        // Optional: break early for this 'i' as BFS guarantees shortest
                    }
                }
            }
        }
        return minCycle == Integer.MAX_VALUE ? -1 : minCycle;
    }
}
```
