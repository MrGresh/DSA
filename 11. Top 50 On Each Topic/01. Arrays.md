## 🛠 Useful Patterns to Remember

### 1. Two Pointers
Used for searching pairs in sorted arrays or partitioning elements.


### 2. Sliding Window
Essential for problems involving contiguous subarrays or finding the longest/shortest window that satisfies a condition.


### 3. Prefix Sum / Suffix Sum
Allows $O(1)$ range sum queries after $O(n)$ precomputation. Useful for balance point problems.


### 4. Kadane’s Algorithm
The optimal way to find the maximum subarray sum in linear time.
$Local\_Max[i] = max(A[i], A[i] + Local\_Max[i-1])$

### 5. Boyer-Moore Voting Algorithm
A clever way to find a majority element in $O(n)$ time and $O(1)$ space.

### 6. Cycle Sort Pattern
Used when the array contains numbers in a specific range (like 1 to N) to find missing or duplicate values.
