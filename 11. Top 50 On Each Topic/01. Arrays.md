# 🚀 Array Mastering Roadmap

A curated list of array problems categorized by difficulty. Perfect for technical interview preparation and sharpening algorithmic thinking.

## 🟢 Easy Problems
- [ ] **Second Largest Element** (Find the second maximum without sorting)
- [ ] **Third Largest Element**
- [ ] **Reverse an Array** (In-place reversal)
- [ ] **Reverse Array in Groups** (Reverse every k elements)
- [ ] **Rotate Array** (Left/Right rotation by D positions)
- [ ] **Three Great Candidates** (Maximum product of three numbers)
- [ ] **Max Consecutive Ones**
- [ ] **Move All Zeroes To End** (Maintaining relative order)
- [ ] **Wave Array** (a1 >= a2 <= a3 >= a4...)
- [ ] **Plus One** (Increment a number represented as an array)
- [ ] **Stock Buy and Sell – One Transaction** (Kadane's variation)
- [ ] **Stock Buy and Sell – Multiple Transactions**
- [ ] **Remove Duplicates from Sorted Array**
- [ ] **Alternate Positive Negative** (Maintaining order)
- [ ] **Array Leaders** (Elements greater than all elements to their right)
- [ ] **Missing and Repeating in Array**
- [ ] **Missing Ranges of Numbers**
- [ ] **Sum of all Subarrays** (O(n) approach using contribution technique)

## 🛠 Useful Patterns to Remember
* **Two Pointers:** Used for sorted arrays, reversing, and partitioning.
* **Sliding Window:** Essential for subarray problems with constraints.
* **Prefix Sum:** Great for range sum queries.
* **Kadane’s Algorithm:** The gold standard for maximum subarray sums.
* **Dutch National Flag:** For sorting arrays with 0s, 1s, and 2s.
