## 🛠 Useful Patterns to Remember

### 1. Layer-by-Layer Traversal
Used for rotation and spiral problems. You process the outer boundary, then move to the inner $(N-2) \times (N-2)$ matrix.

### 2. Transpose and Flip
A common trick to rotate a matrix by 90 degrees:
* **Step 1:** Swap $ matrix[i][j] $ with $ matrix[j][i] $ (Transpose).
* **Step 2:** Reverse each row for clockwise, or reverse each column for counter-clockwise rotation.

### 3. BFS / DFS on Grids
Essential for "Island" and "Pathfinding" problems. Treat each cell $(r, c)$ as a graph node connected to its neighbors (Up, Down, Left, Right).


### 4. 2D Prefix Sum
Useful for finding the sum of any sub-rectangle in $O(1)$ time after $O(R \times C)$ preprocessing.

### 5. Flattening the Matrix
For searching in sorted matrices, you can treat a sorted $M \times N$ matrix as a single sorted array where $index \to (index / N, index \% N)$.

### 6. DP on Grids
Used for pathfinding (Min/Max cost). Usually follows the relation:
$DP[i][j] = Matrix[i][j] + min(DP[i-1][j], DP[i][j-1])$
